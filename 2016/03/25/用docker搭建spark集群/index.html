用docker搭建spark集群

  • docker
  • spark

第一步 从docker官方仓库下载docker镜像

docker pull epahomov/docker-spark

第二步 启动master

第三步 启动worker

配置master-ip 172.17.0.4 为本机ip，172.17.0.3为master ip

启动worker
./start-worker.sh

可以启动多台worker

打开master：8080 可以查看集群部署情况

在master 中打开spark-shell 就可以进行编程了
